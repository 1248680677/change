欢迎
------------

   Dice!是一款基于酷Q,采用 **C++** 编写的掷骰插件,编译环境为 **VS2017** 。

   如果您不了解酷Q是什么，请访问:`酷Q官网 <http://cqp.cc/>`_  

   如果您想要查看源代码:请访问:`Github <http://github.com/w4123/Dice/>`_  

   如果您想要直接使用掷骰服务:请加机器人QQ3579798350或1525964820(推荐)

   如果您想要了解更多信息，请加入QQ群624807593
