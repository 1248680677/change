前言
-----------

   由于多方面原因，为了保证服务的稳定进行，在群中使用Dice!机器人云掷骰服务需要进行申请，同时，我们也开放了受信任用户的申请，具体内容如下


申请群中使用权限
-----------------------

    请先 **将骰娘拉入群中** 再 **提交申请** ,申请方法有两种:

    1.加入群624807593，在群中发布消息，格式为"入群申请:群号:[群号] 原因:[原因] 我已仔细阅读并保证遵守Dice!云掷骰服务用户协议"（推荐）

    2.私聊机器人，输入"!info 入群申请:群号:[群号] 原因:[原因] 我已仔细阅读并保证遵守Dice!云掷骰服务用户协议"，注意,!info和入群申请之间必须有一个空格。

    一般来说，管理员会在12小时内处理你的请求，因此，除非上一次有信息填写错误等原因，在12小时内请不要重复申请。申请成功后，申请成功的提示会被发送到群中，如果申请失败或者需要更多信息，管理员则会联系你(如果管理员联系你的方式是通过机器人，在回复信息时需要使用!info+空格+你想说的话)。如果12小时后，申请仍没有被处理则可以再次申请。


申请受信任用户
-------------------------

    受信任用户邀请机器人进群后无需申请即可使用机器人。

    受信任用户的申请条件:
    
    1.使用本机器人至少两周，且在这两周中保持相对活跃的状态

    2.没有违反Dice!云掷骰用户协议的行为

    3.通过管理员审核

    申请方法有两种:

    1.加入群624807593，在群中发布消息，格式为"受信任用户申请: 原因:[原因]"（推荐）

    2.私聊机器人，输入"!info 受信任用户申请: 原因:[原因]"，注意,!info和入群申请之间必须有一个空格。

    一般来说，管理员会在12小时内处理你的请求，因此，除非上一次有信息填写错误等原因，在12小时内请不要重复申请。申请成功后，申请成功的提示会被发送到群中，如果申请失败或者需要更多信息，管理员则会联系你(如果管理员联系你的方式是通过机器人，在回复信息时需要使用!info+空格+你想说的话)。如果12小时后，申请仍没有被处理则可以再次申请。
   
    注:由于受信任用户拥有较大的权利，我们只会限量发布受信任用户的名额。并且一旦受信任用户违反了Dice!云掷骰服务用户协议，不仅会受到和普通用户一样的惩罚，还会被剥夺受信任用户的身份。
